# AI Surveillance System

Real-time AI surveillance system for automatic person detection using YOLOv8.

## Features

- Real-time person detection
- YOLOv8 object detection
- OpenCV video processing
- Telegram photo alerts
- Telegram video alerts
- Flask live video streaming
- Automatic alert cooldown
- Buffer recording before alert

## Technologies

- Python
- YOLOv8
- OpenCV
- Flask
- Requests
- NumPy
- Ultralytics

## Project Structure

```
project/
```

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Configuration

Replace the Telegram Bot Token and Chat ID with your own credentials.
